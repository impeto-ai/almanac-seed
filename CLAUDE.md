# CLAUDE.md (shim)

Este repositorio usa o padrao aberto `AGENTS.md` como fonte de verdade.

**Antes de qualquer acao, leia, nesta ordem:**
1. `SEED.md` — a receita do software (o que construir + criterios de aceite + build loop).
2. `AGENTS.md` — regras do construtor (stack, seguranca, Definition of Done).
3. `rules/*.md` — regras detalhadas, se existirem.

Siga `AGENTS.md` integralmente. Nao declare a tarefa concluida ate o Definition of Done
(Secao 16 da SEED) passar inteiro, com deploy de producao verificado.
